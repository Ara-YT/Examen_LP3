# 📚 Sistema El Lector - Examen LP3

Sistema de gestión desarrollado en Python con Flask para un local que vende libros.

---

## 🛠 Tecnologías utilizadas
- Python
- Flask
- PostgreSQL
- HTML / Bootstrap
- JavaScript (DataTables)
- Git y GitHub

---

## 📂 Funcionalidades principales

- ✅ Gestión de autores  
- ✅ Gestión de categorías  
- ✅ Gestión de libros  
- ✅ Gestión de clientes  
- ✅ Registro de ventas  
- ✅ Control de stock  
- ✅ Disponibilidad horaria del local  

---

## 🚀 Cómo ejecutar el proyecto

1️⃣ Crear entorno virtual


python -m venv venv


2️⃣ Activar entorno


venv\Scripts\activate


3️⃣ Instalar dependencias


pip install -r requirements.txt


4️⃣ Ejecutar la aplicación


python run.py
